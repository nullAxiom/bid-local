# BidLocal 🦞
### Transparent contractor marketplace where bids are public — with your name attached.

## The Problem

Getting contractor quotes is broken:
- Prices vary wildly with zero transparency
- Scammers lowball to win, then overcharge
- Homeowners have no idea what's fair
- Contractors can't differentiate on quality — only price

## The Proof (Real Experiment, Feb 2026)

Posted ONE Facebook question asking what grading a yard should cost.

**Result in 24 hours:**
- 15+ contractor responses with PUBLIC pricing
- Market rate established instantly ($300 manual → $2,500 with equipment)
- Contractors competed on value, not just price
- Homeowner got 3 quotes same day

**The accountability loop worked** — contractors gave real numbers because their name was attached publicly.

## The Solution: BidLocal

A marketplace where:
- **Bids are public** — attached to the contractor's name and reputation
- **History is visible** — see what a contractor charged on past jobs
- **Accountability is built in** — you can't ghost a public bid
- **Reviews are verified** — tied to actual completed jobs

## TradeApp (The Field Tool)

Voice-first logging for contractors on the job:

> "Just installed 3 PTAC units at Candlewood Suites, Augusta GA. 4 hours labor, parts $340."

That becomes:
- An invoice
- A portfolio entry
- A bid reference for future jobs
- A verified review anchor

## Business Model

- **Free** for homeowners to post jobs and see bids
- **$9-19/month** TradeApp subscription for contractors (voice logging, invoicing, bid history)
- **Small % fee** on completed jobs (optional, marketplace model)

## The Stack

- Built on OpenClaw (open source AI agent framework)
- Voice logging via ElevenLabs + Twilio
- Web scraping + market data via Firecrawl
- Transparent bid database
